### 程序架构

聊天室分为四个部分，前端，后端，qt客户端，c++服务器端。

### 功能

1. 前端：用户注册，应用下载。

2. 后端：登录验证，前端数据传入数据库。

3. qt客户端：登录界面，聊天界面。

4. c++服务器端：数据转发，通信格式的处理。
