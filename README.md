### 概述

通过 QT 编写应用程序通信软件，可一对一和一对多通信。

### 技术
- html
- css
- QT
- C++
- Boostrap

### 功能

- Web 前端
  - 用户注册
  - 应用下载

- Web 后端
  - 登录验证
  - 前端数据传入数据库

- QT 客户端
  - 登录界面
  - 聊天界面

- C++ 服务器端
  - 数据转发
  - 通信格式的处理
